# SEE10 Image Naming

Folder:
- `docs/assets/images/see10/`

Recommended filename format:
- `see10-{topic}-{step:00}.png`
- `see10-{topic}-{step:00}-{variant}.png`

Rules:
- use lowercase English letters
- use hyphens (`-`) between words
- keep topic short and stable
- step number uses 2 digits: `01`, `02`, `03`
- use `.png` by default for screenshots
- use `-dark`, `-light`, `-annotated`, `-error` only when a variant is needed

Examples:
- `see10-runtime-start-01.png`
- `see10-runtime-start-02.png`
- `see10-camera-swap-01.png`
- `see10-output-folder-01-annotated.png`
- `see10-troubleshooting-01-error.png`
