# 光學量測


## 硬體設置

* **CS-LE005-D**
  
參數設置:
|   Parameter   |   Details   |
|   ----   |   ----   |
| Communication method   | Start/Stop synchronization; Half duplex      |
|   Baud rate          | 115200 bps (fixed) |
|   Character length   | 8 bits |
|   Parity              | None |
|   Stop bits           | 1 bit |
|   Delimiter Code      | --- |

> 請連接到 SuperCarter No.30



* **Chroma Meter CL-200A 通訊配置**
  
參數設置:
|   Parameter   |   Details   |
|   ----   |   ----   |
| Communication method   | Start/Stop synchronization; Half duplex      |
|   Baud rate          | 9600 bps (fixed) |
|   Character length   | 7 bits |
|   Parity              | Even |
|   Stop bits           | 1 bit |
|   Delimiter Code      | CR+LF |
> 請連接到 SuperCarter No.31
